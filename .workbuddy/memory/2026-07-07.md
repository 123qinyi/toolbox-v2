## 修复人员管理表格末行被裁切不可见
- **根因**：StaffManager.tsx 表格滚动区设了 `max-h-[60vh]`（相对视口固定值），当弹窗内统计面板占空间多时，父级 flex 分配高度 < 60vh，滚动区超出父级部分被 `overflow-hidden` 裁切，末行（欧仁强）不可见
- **修复**：`max-h-[60vh]` → `h-full`，让滚动区跟随父级 flex 高度，消除固定上限与 flex 布局的冲突
- 教训：flex 布局中子元素不应设独立的视口相对高度上限，应让 flex 分配高度生效
